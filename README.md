# TecladoEnPantalla

## Fecha
* 2012

## Descripcion del proyecto
* Permite utilizar el programa de lavadoras tonello con una pantalla touchscreen.

## Herramientas utilizadas
* Visual Basic 6
* WinSpy++
* dsmouse.dll

## Modelos de PLC, etc.
* Pantalla SAW touchscreen
